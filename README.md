# WhytChat

WhytChat is a desktop application built with Tauri, React, and Rust that provides an AI-powered chat interface with advanced features like debate modes and multi-agent orchestration.

## Features

- Multi-agent debate system
- Local LLM integration
- Cross-platform desktop app (Windows, macOS, Linux)
- Modern React frontend with Tailwind CSS
- Secure Rust backend with Tauri

## Development

### Prerequisites

- Node.js 18+
- Rust stable
- Tauri CLI

### Setup

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
```
